# Data License — CC0 (Public Domain)

## What this covers

All **community-contributed data** in this repository is released under [CC0 1.0 Universal (Public Domain Dedication)](https://creativecommons.org/publicdomain/zero/1.0/).

This includes:

- **Genre taxonomy** — genre names, descriptions, and hierarchical relationships
- **Genre connections** — directed edges between genres (influenced_by, evolved_into, etc.)
- **City-to-genre mappings** — which cities are associated with which musical scenes
- **Biome assignments** — mapping genres to visual terrain types
- **Coordinate data** — UMAP-projected positions for the Genre World layout

## Why CC0?

Music genre knowledge belongs to everyone. By using CC0:

- Anyone can build on this data without restriction
- Other music projects can reuse the taxonomy freely
- Academic researchers can cite and extend it
- No license compatibility headaches

## What this does NOT cover

- **Application code** — licensed under [AGPL-3.0](LICENSE)
- **Discogs data** — already CC0 from Discogs ([their terms](https://www.discogs.com/developers/#page:database,header:database-data-dumps))
- **Ishkur's Guide data** — included for research/educational reference; see [ishkur.com](https://music.ishkur.com/) for original terms
- **Third-party assets** (fonts, icons, textures) — see individual licenses in their directories

## Contributing data

By submitting genre corrections, city additions, track suggestions, or taxonomy edits to this project, you agree to release your contribution under CC0.
